# Prompack Predictive KPI Dashboard

Script para análisis predictivo de KPIs utilizando Python y scikit-learn.

## Funcionalidades

- Análisis de hojas: Ene_26, Feb_26, Mar_26
- Predicción de rendimiento de KPIs
- Comparación con metas 2026
- Generación automática de reportes

## Instalación

```bash
pip install -r requirements.txt
